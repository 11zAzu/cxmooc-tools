# 关于超星慕课小工具

cxmooc-tools，一个可以辅助观看超星慕课的浏览器扩展，支持自动观看，自动答题等功能。兼容主流浏览器以及Tampermonkey脚本。

# 浏览器适配列表

  • Chrome for PC
  • Firefox for PC
  • Firefox for Mobile
  • QQ 浏览器 for PC
  • Tampermonkey

# 功能支持列表

  • 视频挂机
  • 视频秒过
  • 视频倍速
  • 静音播放
  • 自动答题
  • 自动阅读
  • 自动填写验证码
